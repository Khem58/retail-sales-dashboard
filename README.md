# Retail Sales Dashboard Project

## Overview
This project analyzes retail sales data using Power BI and Excel.

## Dataset
The dataset includes:
- Sales
- Profit
- Orders
- Products
- Regions

## Tools Used
- Excel
- Power BI
- GitHub

## Key Insights
- Top performing products identified
- Regional sales comparison
- Monthly sales trends
- Profitability analysis

## Files Included
- Excel dataset (.xlsx)
- Power BI dashboard (.pbix)

## Goal
To practice data analysis and build a business intelligence dashboard for portfolio use.
